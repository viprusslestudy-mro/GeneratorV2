# Как пользоваться DevModePanel

Меню пока не менялось – в текущей реализации панель разработчика (DevModePanel) появляется автоматически, если в листе ⚙️ APP_SETTINGS включён флаг

```
developer_mode_search_translation   ON
```

## Как пользоваться

### Включите режим разработчика

Откройте Google‑таблицу, лист ⚙️ APP_SETTINGS.
Найдите строку developer_mode_search_translation и установите значение ON (или TRUE).
Сохраните лист.
Обновите дашборд в браузере

После сохранения листа перезагрузите страницу с React‑дашбордом (или выполните clasp push → clasp open → обновите окно).
В правом‑нижнем углу появится плавающая панель 🔍 Dev Mode: Непереведённые фразы с счётчиком.

### Сбор непереведённых фраз

Перемещайтесь по всем вкладкам дашборда (Finance, Channels, Support Stats, Support Tags).
Каждый раз, когда хук useTranslation не находит перевод для ключа, он автоматически добавит этот ключ во внутреннее множество missingTranslations.
Панель покажет текущее количество собранных фраз.

### Отправка в таблицу переводов

Нажмите кнопку «📤 Добавить в таблицу (N)」 на панели.
Система отправит все собранные ключи на бэкенд, где они будут добавлены в лист 🌐 TRANSLATIONS с пометкой ___AUTO_MISSING_<секция>_<дата>.
В колонке RU будет записан сам ключ (placeholder), а колонка EN останется пустой – вы сможете заполнить её вручную.

### Очистка списка

Если хотите начать сбор заново, нажмите кнопку «🗑️ Очистить» – внутреннее множество сбросится.
